# CRM - CRM Management Dashboard

A modern and responsive CRM (Customer Relationship Management) Dashboard built using **HTML, CSS, and JavaScript**. The application provides a complete CRM workflow experience with lead management, sales pipeline tracking, customer management, task monitoring, analytics, and reporting.

## 🚀 Live Demo : [Live Demo](basiccrmdashboard.netlify.app)
---

## 📌 Features

### Dashboard

* KPI cards with business metrics
* Revenue and lead trend charts
* Lead source analytics
* Management insights section
* Team performance overview
* Activity feed
* Conversion funnel visualization

### Leads Management

* View and manage all leads
* Lead stages tracking
* Search and filter leads
* Export lead data to CSV
* Add new leads through modal forms

### Contacts Management

* Organized contact directory
* Customer and prospect categorization
* Contact information overview

### Sales Pipeline

* Visual pipeline board
* Opportunity tracking across stages
* Deal progress monitoring

### Customers

* Customer performance metrics
* Revenue tracking
* Retention monitoring

### Task Management

* Create and manage tasks
* Mark tasks as completed
* Overdue task tracking
* Task statistics dashboard

### Reports & Analytics

* Revenue reports
* Team performance analytics
* Lead source analysis
* Business performance insights

### Additional Features

* Dark / Light theme toggle
* Responsive design
* Interactive charts using Chart.js
* Real-time notifications
* Global search functionality
* Mobile-friendly navigation

```bash

No additional setup or dependencies are required.
